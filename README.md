## Project setup
Follow these steps to serve this project on your local computer.

*Required:* Node.js installed on your device

1. Open up this project in a code editor or IDE.
2. Open up the terminal and run the following command: 
```
npm install
```
3. When the installation is complete, run this second command:
```
npm run serve
```
4. Open up your web browser to http://localhost:8080/. Note, your terminal might tell you differently.
5. Open up the developer inspect tool. If you use google chrome you can right click and click inspect towards the bottom or use the following hotkey combination:
```
ctrl + shift + i
```
6. Once the inspector is open, click the toggle device toolbar button in the upper left corner of the inspector window or use the following hotkey combination:
```
ctrl + shift + m
```
7. Continue to select Pixel 2 XL from the dropdown menu which normally is set to responsive. This can be found on the far left on the device toolbar that comes up after you finished step 6.
